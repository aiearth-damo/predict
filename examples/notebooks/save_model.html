模型保存¶

模型是指机器学习的产物，它封装了算法和学习参数。一旦经过训练，模型就可以用来对新的、看不见的数据进行预测。BentoML提供了一个本地Model Store来保存和管理模型，它是BentoML维护的本地文件目录。本文档解释了BentoML中模型的关键操作

云平台提供了模型存储和版本管理的能力，可以

version = Version(test_version_id)
ckpt = ModelCheckpoint.from_local_path(
    path="/home/data/modelhub/tianxun_landcover_v1.6_20220113_sim.onnx",
    image_size=1024,
    bound=128,
    attached_files={
        "tianxun_landcover_v1.6_20220113_sim.TeslaT4.trt": ModelAttachedFile(
            path="/home/data/modelhub/tianxun_landcover_v1.6_20220113_sim.TeslaT4.trt",
            accelerator_type=AcceleratorType.t4,
        )
    },
)
model_tag = version.save_model(ckpt)
