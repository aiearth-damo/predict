=============
概览
=============

AI Earth Predict 是时空数据和AI模型相结合的开发套件，用于进行大规模遥感数据模型推理的统一计算库，并可以用来并行处理遥感数据，支持本地模式运行，以及将模型部署到 `AI Earth地球科学云平台 <https://engine-aiearth.aliyun.com/#/>`_
